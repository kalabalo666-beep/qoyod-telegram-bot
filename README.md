# بوت تيليجرام قيود - شركة رائد

## نظرة عامة
بوت تيليجرام متكامل لإدارة الفواتير والقيود المحاسبية من ملفات Google Drive وتسجيلها مباشرة في نظام قيود.

### المميزات الرئيسية
- 📱 واجهة تيليجرام سهلة وعربية 100%
- 📁 دعم Google Drive لاستقبال الملفات
- 📄 معالجة تلقائية للفواتير والقيود
- 💾 تكامل كامل مع Qoyod API
- ✅ تسجيل الفواتير والقيود بشكل آلي
- 📊 تقارير وإحصائيات

### التثبيت

```bash
pip install -r requirements.txt
```

### التشغيل

```bash
python main.py
```
